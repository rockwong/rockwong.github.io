Git 使用教程

这是一份面向初学者的 Git 入门指南，涵盖日常开发中最常用的命令和工作流。

基础配置

安装 Git 后，首先设置你的身份信息——每次提交都会引用这些数据：

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

创建仓库

在项目目录中初始化一个新的 Git 仓库，或者克隆一个已有的远程仓库：

git init
git clone https://github.com/user/repo.git

日常工作流

查看状态

git status

暂存文件

git add filename
git add .          # 暂存所有变更

提交变更

git commit -m "描述你的变更"

推送到远程

git push origin main

拉取最新代码

git pull

分支操作

分支让你可以在不影响主线的情况下开发功能或修复 bug：

git branch feature-login    # 创建分支
git checkout feature-login   # 切换到分支
git checkout -b feature-api  # 创建并切换
git merge feature-login      # 合并分支到当前分支
git branch -d feature-login  # 删除已合并的分支

常用技巧

  • git log --oneline --graph — 查看简洁的提交历史图
  • git stash / git stash pop — 暂存当前工作，稍后恢复
  • git reset --soft HEAD~1 — 撤销上一次 commit，保留修改
  • git diff — 查看尚未暂存的变更
