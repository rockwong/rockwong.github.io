Sass Mixin Library

A small, focused collection of Sass mixins I use across projects. Each one solves a recurring problem without pulling in a full framework.

Clearfix

When floats need containing, the classic clearfix — still useful in legacy contexts:

@mixin clearfix {
  &::after {
    content: "";
    display: table;
    clear: both;
  }
}

Media Queries

Named breakpoints make responsive code easier to scan at a glance:

$break-sm: 576px;
$break-md: 768px;
$break-lg: 1024px;

@mixin respond-above($bp) {
  @media (min-width: $bp) { @content; }
}

@mixin respond-below($bp) {
  @media (max-width: $bp - 1) { @content; }
}

Ellipsis

Truncate text that overflows its container with a single line:

@mixin ellipsis {
  overflow: hidden;
  white-space: nowrap;
  text-overflow: ellipsis;
}

Center

Centering with flexbox, both axes or just one:

@mixin center($axis: both) {
  display: flex;
  @if $axis == both {
    justify-content: center;
    align-items: center;
  } @else if $axis == x {
    justify-content: center;
  } @else if $axis == y {
    align-items: center;
  }
}

Typography Scale

A simple modular scale for consistent type sizing:

@mixin type-scale($step) {
  $base: 16px;
  $ratio: 1.25;
  font-size: $base * pow($ratio, $step);
}
