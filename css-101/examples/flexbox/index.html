Flexbox Examples

Flexbox is the go-to layout model for one-dimensional arrangements. Here are a few practical patterns with code.

Evenly Spaced Navigation

A classic nav bar with items distributed evenly across the available space:

Home
About
Blog
Contact
.nav {
  display: flex;
  justify-content: space-between;
  gap: 8px;
}

Centered Card

Perfect centering — the use case that justified flexbox for a generation of developers:

.card-container {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 200px;
}

Equal-Height Columns

Columns that stretch to match the tallest sibling, no JavaScript required:

.columns {
  display: flex;
  align-items: stretch;
}
.column { flex: 1; }
